# AWS Cognito User Authentication Workshop

#### Overview

**AWS Cognito** provides user identity and access management for web and mobile applications. It enables you to add user sign-up, sign-in, and access control to your applications quickly and easily.

In this workshop, you will learn how to:
- Create and configure a Cognito User Pool
- Setup password policies and email verification
- Configure App Client with Hosted UI
- Implement basic email/password authentication
